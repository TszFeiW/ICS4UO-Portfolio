Unit 1 - OOP

Objects and Classes

A software object is a location in memory that corresponds to a value. They can have methods, which allows them to perform actions when run.

A Java class is a blueprint for an object. Java comes with library classes such as Java.lang.math, but classes can also be created by the user.

Running the Java application creates objects of classes and allows for use of the methods within the objects.

For example, one dolphin ice mold class could be used as the template to create many dolphin ice cube objects.

A static characteristic in a class is a variable that is shared across all objects of the class; there is only 1 copy of it. This is useful when the variable is a common property, for example the maximum class size in a university.
